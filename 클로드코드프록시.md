## Free Claude Code 

> https://github.com/Alishahryar1/free-claude-code

- Use Claude Code CLI, VS Code, JetBrains ACP, or chat bots through your own Anthropic-compatible proxy.
- Free Claude Code routes Anthropic Messages API traffic from Claude Code to NVIDIA NIM, OpenRouter, DeepSeek, LM Studio, llama.cpp, or Ollama.
- It keeps Claude Code's client-side protocol stable while letting you choose free, paid, or local models.
